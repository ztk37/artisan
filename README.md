## Commands

| Command               | Operation                                                 |
| :-------------------- | :-------------------------------------------------------- |
| `artisan help`        | Print this message or the help of the given subcommand(s) |
| `artisan init`        | Initialise artisans home directory                        |
| `artisan new`         | Create a new project from a template                      |
| `artisan config list` | List all config values                                    |
| `artisan config get`  | Get a config value by key                                 |
| `artisan config set`  | Set a config value by key                                 |
